# Tech News Android App

Android application to display latest technology news using Public API.

## Features
- Latest tech news
- RecyclerView news list
- Thumbnail images
- Open article in browser
- Responsive UI

## Tech Stack
- Kotlin
- Retrofit
- Glide
- RecyclerView
- CardView
